# HTTP client/server pair in Bash with Netcat

## Description
This directory contains two Bash scripts using Netcat to emulate an HTTP client/server pair.


## Usage
1. Run
   ```
   ./server.sh
   ```
   to start up the HTTP server on your local machine

2. In a separate shell, run
   ```
   ./client.sh
   ```
   to send HTTP requests to the server

3. Watch the server respond to the client's requests on the client's terminal
